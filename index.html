apps.0install.net provides a repository of common tools, libraries and runtime environments:

  • 0install tools
  • Python

  • Ruby
  • JavaScript
  • Protobuf tools
  • OCaml
  • Golang
  • Java
  • .NET
  • Libraries

Feeds can be hosted on any normal website. 0mirror provides a list of all currently known feeds.

If you want to create your own feeds to make additional apps available via Zero Install, please read the packaging guide.
