The JDK feed automatically selects an OpenJDK distribution based your operating system:

  • Eclipse Temurin JDK on Linux and macOS
  • Microsoft JDK on Windows
  • OpenJDK as a fallback

You can explicitly control which OpenJDK distribution is used by referencing the relevant feed directly.
